### Install with npm or yarn

heliosRX requires bolt-compiler as a peer dependency, so please run:

```bash
npm install --save heliosrx
npm install -g bolt-compiler
```

or, if you're using yarn:

```bash
yarn add heliosrx
yarn add -g bolt-compiler
```
